Для работы проекта требуется добавить API ключи в файл .env

1. VITE_API_KEY_FIREBASE - ключ от базы данных Firebase (для работы с данными и авторизации в приложении)
2. VITE_MWS_API_KEY - ключ от MWS GPT (для работы с функционалом ИИ)
3. VITE_MWS_AI_MODEL - идентификатор деплоймента модели ИИ

Ключи и идентификатор модели ИИ можно получить по запросу на почтовый ящик daniillanin@yandex.ru

Для запуска проекта введите в терминале:
1.  git clone https://github.com/daniillanin/mwstables_project_case.git mwstables_project_case
2.  cd mwstables_project_case
3.  npm install
4.  npm run dev
5.  открыть в браузере http://localhost:5173
